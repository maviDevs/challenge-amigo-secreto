# Amigo Secreto 🎁  

Este é um projeto simples de sorteio de **Amigo Secreto**, onde os usuários podem adicionar nomes e sortear aleatoriamente um amigo.  

## 📂 Estrutura do Projeto  

- **index.html** → Estrutura da página.  
- **style.css** → Estilos visuais.  
- **app.js** → Lógica do sorteio e manipulação da lista de amigos.  

## 🚀 Como Usar  

1. Abra o arquivo `index.html` no navegador.  
2. Digite os nomes dos participantes no campo de entrada.  
3. Clique em **Adicionar** para incluir um nome.  
4. Após adicionar os nomes, clique em **Sortear Amigo** para realizar o sorteio.
